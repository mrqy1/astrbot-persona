# AstrBot 人格模板

一个专为 AI 角色扮演设计的人格模板，基于心理学框架，帮助创建有深度的角色。

## 目录结构

```
astrbot-persona/
├── template.md          # 空白模板，直接复制使用
├── examples/            # 示例人格
│   └── 小泉妃爱/        # 示例角色
│       ├── README.md    # 角色介绍
│       └── persona.md   # 已填写的人格文件
└── README.md
```

## 快速开始

### 方式一：使用模板创建新人格

1. 复制 `template.md` 文件
2. 按模块填写内容
3. 保存为你的角色名称

### 方式二：参考示例

查看 `examples/` 目录下的示例角色，了解如何填写。

## 模板特点

- **心理学基础**：依恋类型、防御机制、认知偏差等专业维度
- **AI 友好**：对话风格、场景反应指南，AI 可直接使用
- **模块化**：每个部分独立，可按需选用
- **填写式**：空白模板，直接复制填写
